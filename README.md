Face Mask Detection 

About
This project detects whether a person is wearing a mask or not using Python and OpenCV.


Technologies
* Python
* OpenCV
* TensorFlow / Keras


How to Run

1. Install requirements
  tensorflow
openCV (cV2)
download the face detection configaration "deploy.prototxt"
and
pretrained  model of face detection DNN("res10_300x300_ssd_iter_140000.caffemodel") 

3. Run the program
python Detect.py


Files
* Detect.py → run detection
* Train.py → train model
* deploy.prototxt → face detection config
* caffemodel → face detection model

Model & Dataset
Dataset and trained model are not uploaded because of large size.

You can:
Train model using Train.py
Or 
Add your own model file
